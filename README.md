# LightMVC

LightMVC is a pure php framework that only contains model-view-controller components; Simplicity is the priority of the project.

## Usage

### Directory Structure

```
+ public
    | - htaccess
    | - index.php
+ config
    | - config.ini.php
+ application/
    | - bootstrap.php
    | - module
        | + example
            | + controller
                | - Index.php
            | + template
                |+ index   
                    | - index.tpl
+ library
+ tests
```
